# Direction

## Visual Spine

- Palette: `#080808`, `#0d0d0d`, `#1a1a1a`, `#333333`, `#ff6b4a`, `#ff8a70`, `#e5e5e0`.
- Type: Inter for display, Geist for UI copy, JetBrains Mono for metadata.
- Layout: Full-page 70/30 auth screen. The left 70% is dedicated to a clean LineWaves visual panel with bottom-left copy; the right 30% contains the compact auth controls. Use the full viewport width instead of compact centered containers.
- Components: Hairline panels, Lucide icons on the dark background, breathable auth controls.

## Signature Moment

The single signature moment is the React Bits LineWaves layer owning the left half of the page. It should not sit behind text.
