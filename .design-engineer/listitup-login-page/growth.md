# Growth

## Primary CTA

`Sign in`

## Secondary Actions

- Email magic link
- Sign up mode toggle
- Reset password

## Friction Audit

- Email and password remain the only required inputs on the default path.
- Sign-up stays in the same surface and adds only Display Name.
- Password recovery is adjacent to the password label.
- Magic link is visible but visually subordinate to the primary sign-in button.
- The removed right-side preview keeps attention on the login flow and reduces scan competition.

## Conversion Bet

A focused single-surface login with one primary CTA should increase completed sign-ins by reducing distraction around the form.
